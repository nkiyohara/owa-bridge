Local-first Outlook Web tooling

Mail and calendar,

A CLI and MCP server that uses your interactive Outlook Web session. No Microsoft Graph app, hosted relay, or password capture.

What it does

Mail

List and search messages, read bodies, save drafts, send, move, and change read state.

Calendar

List, create, update, and cancel events with required or optional attendees.

Teams links

Create a reviewed Outlook calendar meeting with a Teams join link.

For AI agents

Twenty narrow MCP tools expose the same typed operations and stable JSON as the CLI. Consequential writes use an exact preview and commit flow. 

owa mcp setup codex
codex mcp get owa

owa mcp setup claude-code
claude mcp get owa

MCP guide · JSON contract

Safety by default

  • Authentication stays visible, interactive, and browser-owned.
  • Writes require a server-enforced preview and commit.
  • Versioned items use exact IDs and change keys.
  • Ambiguous writes fail closed instead of retrying automatically.

General Teams access, Graph passthrough, hosted relays, unattended login, tenant-wide access, and permanent delete are out of scope.
